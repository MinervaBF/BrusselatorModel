# BrusselatorModel

This repository contains code and results related to the simulation of the Brusselator Model for Physics of Complex Systems in the University of Granada.

The file GenerationBrusselator.py contains the code to generate the simulation.

The files AnimationBrusselator2D.py and AnimationBrusselator3D.py contain the code to generate animations from the data simulated.

The file Graphs.py allowed us to represent $Re(\sigma)$ vs $q$. This graphs allowed us to predict the behaviour of the system.
